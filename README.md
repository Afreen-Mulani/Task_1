# Task 1: Responsive Landing Page

## Objective
Created a simple responsive landing page using HTML and CSS.

## Features
- Header & navigation bar
- Hero section with call-to-action
- Menu section using grid layout
- About section
- Responsive design using media queries
- Mobile-friendly navigation bar

## Tools Used
- HTML5
- CSS3
- VS Code
- Live Server
- GitHub

## How to Run
1. Download the code or clone repo
2. Open index.html in a browser
3. Enjoy the landing page!
